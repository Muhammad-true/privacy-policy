Политика конфиденциальности

Это приложение "Steto" (далее — «мы», «нас», «наше») уважает вашу конфиденциальность. Ниже описано, какие данные могут собираться, как они используются и как обеспечивается их безопасность.

1. Сбор данных

Мы не собираем персональные данные без вашего согласия. Приложение может использовать сторонние библиотеки и API (например, аналитика, реклама, Audius API), которые могут собирать обезличенные данные, такие как IP-адрес, тип устройства или действия внутри приложения.

2. Использование Audius API

Наше приложение использует Audius API для получения музыкального контента, метаданных о треках, альбомах и исполнителях. Мы не передаём никакие пользовательские персональные данные в Audius. Вся информация, получаемая от Audius, используется только для отображения и воспроизведения контента.

3. Использование данных

Собранные обезличенные данные используются исключительно для улучшения качества приложения и его функциональности.

4. Передача данных третьим лицам

Мы не передаём ваши данные третьим лицам, за исключением случаев, когда это требуется законом или осуществляется через используемые сторонние сервисы (например, Google Analytics, AdMob, Audius и др.).

5. Безопасность данных

Мы принимаем разумные меры для защиты ваших данных, однако абсолютную безопасность гарантировать невозможно.

6. Доступ для детей

Наше приложение предназначено для широкой аудитории, включая детей. Мы соблюдаем требования COPPA и не собираем личную информацию от детей младше 13 лет без согласия родителей.

7. Изменения политики

Мы оставляем за собой право обновлять настоящую политику конфиденциальности. Все изменения будут опубликованы на этой странице с указанием даты обновления.
